# 特定商取引法に基づく表記（ドラフト・将来の有料化時テンプレ）

> **注意: 弁護士レビュー前のドラフト／テンプレートです。`【】` は事業者が確定して埋める項目。**
>
> **現状、本サービスは無償提供であり、有料サービス・サブスクリプション・物販を行っていないため、本表記の掲載義務は生じていない想定。**
> 将来、有料機能・サブスク・広告課金等を導入する際に、本テンプレを弁護士確認のうえ掲載する。

## 導入時の確認事項（弁護士）

1. サブスクの**自動更新・解約導線**の表示義務（特商法・改正に伴う「契約締結前の確認画面」等）。
2. アプリ内課金（App Store / Google Play）を用いる場合のストア規約と特商法表記の整合。
3. 価格の**税込表示**、無料トライアルからの自動有料転換時の明示。
4. クーリングオフ非適用（通信販売）の整理と、自社都合解約・返金ポリシー。
