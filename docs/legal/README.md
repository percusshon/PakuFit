# PakuFit 法務レビュー用ドキュメント一式

このディレクトリは、PakuFit（パクフィット）を一般公開する前に**弁護士・専門家へレビューを依頼するための資料一式**です。
非専門家（開発側）が事実関係と論点を整理したものであり、**本書自体は法的助言ではありません**。
最終的な文言・適法性判断は弁護士のレビューを前提とします。

## 構成

| ファイル | 内容 | 主な読み手 |
| --- | --- | --- |
| `legal-review-request.md` | レビュー依頼カバー。サービス概要・データフロー・想定適用法令・**質問リスト**・事業者が確定すべき事項 | 弁護士（最初に読む） |
| `personal-data-inventory.md` | 個人データ取扱台帳。取得項目・利用目的・第三者提供・保管先・保持期間の現状整理 | 弁護士・開発 |
| `privacy-policy-draft.md` | プライバシーポリシー正式版ドラフト（個人情報保護法の必要記載に対応） | 弁護士（添削対象） |
| `terms-of-service-draft.md` | 利用規約ドラフト | 弁護士（添削対象） |
| `tokushoho-draft.md` | 特定商取引法に基づく表記ドラフト（**将来の有料化時テンプレ**。現状は無償提供） | 弁護士（課金導入時） |

## 重要な前提（レビュー時に共有すべき事実）

1. 現状は **MVP / ローカル開発段階**。本番公開・有料化・広告配信はまだ行っていない。
2. カロリー/PFC等の栄養値は一貫して **「概算・目安」** として扱い、医療・診断・治療を目的としない（実装・テストで断定表現を機械的に抑止している）。
3. 写真AIによる外部送信は **明示オプトインのときのみ**（既定はローカル擬似推定、外部送信なし）。
4. 本番のホスティング先リージョン（Supabase）と外部AI事業者の確定が、**外国にある第三者への提供（個人情報保護法28条）** の判断に直結する未確定事項。

## 事業者が埋めるべきプレースホルダ（全ドラフト共通）

ドラフト内の `【】` は事業者が確定して埋める。主なもの:

- `【事業者名】` / `【代表者名】` / `【所在地】`
- `【問い合わせ窓口メールアドレス】` / `【個人情報保護方針の問い合わせ窓口】`
- `【本番ホスティング先と保管リージョン】`（例: Supabase / リージョン）
- `【外部AI事業者名と所在国】`（オプトイン送信先。例: OpenAI／米国 等）
- `【準拠法・裁判管轄】`（例: 日本法／東京地方裁判所 等）
- `【サービス開始日】` / `【規約・ポリシーの制定日・最終改定日】`
