# パクフィット / PakuFit

## 概要

パクフィット（PakuFit）は、食事写真・テキスト入力をもとに、1日あたりのカロリー/PFC（たんぱく質・脂質・炭水化物）を概算し、次の食事候補を提案する食事記録アプリのMVPを想定した設計プロジェクトです。

本プロジェクトは、**診断・治療・疾病改善を目的としない一般的な食事記録と行動サポート**を扱います。

## MVP範囲

- ユーザー登録・ログイン
- 食事写真アップロード（入力保存とAI向けデータ化）
- 食事テキスト入力
- AIによる料理名・食材候補推定
- ユーザーによる補正（料理名・量・食べた割合・調理方法）
- カロリー/PFCの概算表示
- 1日の食事記録一覧の閲覧
- 1日の合計カロリー/PFC表示
- ユーザー目標登録
  - 体重管理
  - 食事バランス改善
  - たんぱく質を意識する食事
  - 脂質を控えめにする食事
  - 外食/コンビニ中心でも整える食事
- 食事履歴・時間帯・残りカロリー・不足しがちなPFCをもとにした**次の食事候補提案**
- 候補提示時の安全な説明文
- RLS（行単位）方針の設計
- 商標一次調査メモ

## 安全方針（MVP）

- 「確定値」ではなく、**概算/推定**を明示
- AI推定だけで判断しない（ユーザー補正を必須化）
- 食事候補は「指示」ではなく「候補」として提示
- 「この食事で痩せる」「脂肪が落ちる」等の断定表現を禁止
- 未成年、妊娠中、持病・通院中、摂食障害既往がある場合は安全文言を表示して専門家相談を促す
- 極端な制限や断食・過度減量を推奨しない

## 名称方針
